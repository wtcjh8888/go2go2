# SLAM 建图运行步骤

## 环境要求
- ROS2 Humble
- Livox Mid360s 雷达，IP: 192.168.2.3
- 主机有线网卡 IP: 192.168.2.107

## 步骤0：配置网络（首次使用）

```bash
sudo ip addr add 192.168.2.107/24 dev enp8s0
sudo ip route add 192.168.2.3/32 dev enp8s0 

```

## 步骤1：编译项目

```bash
cd /home/w/C206Go2
source /opt/ros/humble/setup.bash
colcon build --cmake-args -DROS_EDITION=ROS2 -DDISTRO_ROS=humble
```

> 注意：现在统一从 `/home/w/C206Go2` 根目录编译，安装路径为 `/home/w/C206Go2/install/`

## 步骤2：运行建图（需要两个终端）

**终端1 - 启动雷达驱动：**

```bash
source ~/C206Go2/install/setup.bash
ros2 launch livox_ros_driver2 msg_MID360s_launch.py
```

**终端2 - 启动自动补偿 SLAM 建图：**

```bash
source ~/C206Go2/install/setup.bash
ros2 launch fast_lio mapping_auto_tf.launch.py
```

> 启动后保持不动 2-3 秒，等 IMU 初始化完成后再移动。

## 步骤3：保存地图（建图完成后）

```bash
source ~/C206Go2/install/setup.bash
ros2 service call /map_save std_srvs/Trigger
```

地图保存路径：`/home/w/C206Go2/src/FAST_LIO-ROS2/PCD/test.pcd`

## 步骤4：查看地图

### 方法1：pcl_viewer（快速查看）

```bash
pcl_viewer /home/w/C206Go2/src/FAST_LIO-ROS2/PCD/test.pcd
```

- 鼠标左键拖拽：旋转
- 鼠标滚轮：缩放
- 按 `r`：重置视角
- 按 `j`：截图

### 方法2：rviz2 查看 PCD 地图（推荐）

一键启动（自动发布 PCD + 打开 rviz2，已预配置话题）：

```bash
bash ~/C206Go2/src/FAST_LIO-ROS2/scripts/view_pcd.sh
```

> rviz2 会自动加载 `/cloud_pcd` 话题，无需手动添加。

## 配置文件说明

- 雷达配置：`src/livox_ros_driver2/config/MID360s_config.json`
- SLAM配置：`src/FAST_LIO-ROS2/config/mid360.yaml`

### mid360.yaml 关键参数

| 参数 | 说明 |
|------|------|
| `extrinsic_R` | LiDAR-IMU 外参旋转矩阵，补偿 20° 安装前倾 |
| `extrinsic_est_en` | 是否在线估计外参（手动设外参时关闭） |
| `point_filter_num` | 点云过滤，值越大点越稀疏 |
| `max_iteration` | 匹配迭代次数 |
| `blind` | 近距离盲区（米） |
| `map_file_path` | PCD 保存路径 |

### MID360s_config.json 关键参数

| 参数 | 说明 |
|------|------|
| `ip` | 雷达 IP（192.168.2.3） |
| `host_ip` | 主机有线网卡 IP（192.168.2.107） |
| `extrinsic_parameter` | 不用这里修正建图倾斜；使用 MID360S 自带 IMU 时保持默认，安装角由 `mapping_auto_tf.launch.py` 自动发布 `body -> base_link` 补偿 |



## 自动补偿雷达倾斜建图命令



### 1. 清理旧进程



```bash

pkill -f livox_ros_driver2_node || true

pkill -f fastlio_mapping || true

pkill -f static_transform_publisher || true

pkill -f 'ros2 launch livox_ros_driver2' || true

pkill -f 'ros2 launch fast_lio' || true

pkill -x rviz2 || true

ros2 daemon stop

ros2 daemon start

```



### 2. 启动雷达驱动



```bash

cd /home/w/C206Go2

source /opt/ros/humble/setup.bash

source install/setup.bash

ros2 launch livox_ros_driver2 msg_MID360s_launch.py

```



### 3. 启动自动补偿建图



```bash

cd /home/w/C206Go2

source /opt/ros/humble/setup.bash

source install/setup.bash

ros2 launch fast_lio mapping_auto_tf.launch.py

```



### 4. 只命令行测试，不打开 RViz



```bash

ros2 launch fast_lio mapping_auto_tf.launch.py rviz:=false

```



### 5. 延长 IMU 采样时间



```bash

ros2 launch fast_lio mapping_auto_tf.launch.py sample_duration:=3.0

```



### 6. 检查是否只有一套节点



```bash

ros2 node list

ros2 topic info /livox/lidar

ros2 topic info /livox/imu

ros2 topic info /tf

ros2 topic info /tf_static

```
