# Go2 运动控制指令全集

所有运动指令通过 WebRTC 数据通道发送，话题固定为 `rt/api/sport/request`。

---

## 一、Normal 模式指令 (SPORT_CMD)

在 normal / ai 模式下生效。

### 1.1 基础控制（无参数）

| 指令 | api_id | 说明 |
|------|--------|------|
| `Damp` | 1001 | 放松/阻尼模式（机器人软掉） |
| `BalanceStand` | 1002 | 平衡站立 |
| `StopMove` | 1003 | 停止运动 |
| `StandUp` | 1004 | 站起来 |
| `StandDown` | 1005 | 趴下 |
| `RecoveryStand` | 1006 | 摔倒后恢复站立 |
| `Sit` | 1009 | 坐下 |
| `RiseSit` | 1010 | 从坐姿站起 |
| `Standup` | 1050 | 另一种站立 |

```python
await pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["StandUp"]}
)
```

### 1.2 运动控制（带参数）

| 指令 | api_id | 参数 | 说明 |
|------|--------|------|------|
| **`Move`** | **1008** | `{"x": float, "y": float, "z": float}` | **移动：x=前后速度(m/s), y=左右速度(m/s), z=旋转速度(rad/s)** |
| `Euler` | 1007 | `{"roll": float, "pitch": float, "yaw": float}` | 姿态控制 |
| `BodyHeight` | 1013 | `{"data": float}` | 设定身体高度(m) |
| `FootRaiseHeight` | 1014 | `{"data": float}` | 设定抬脚高度 |
| `SpeedLevel` | 1015 | `{"data": int}` | 设定速度等级(1-3) |
| `SwitchGait` | 1011 | `{"data": int}` | 切换步态 |
| `ContinuousGait` | 1019 | 无 | 连续步态模式 |
| `TrajectoryFollow` | 1018 | 轨迹数据 | 轨迹跟踪 |
| `EconomicGait` | 1035 | 无 | 节能步态 |
| `FreeWalk` | 1045 | 无 | 自由行走 |
| `CrossWalk` | 1051 | 无 | 横向行走 |
| `LeadFollow` | 1045 | `{"data": bool}` | 跟随模式开关 |

```python
# 前进
await pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Move"],
     "parameter": {"x": 0.5, "y": 0, "z": 0}}
)

# 后退
await pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Move"],
     "parameter": {"x": -0.5, "y": 0, "z": 0}}
)

# 左移
await pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Move"],
     "parameter": {"x": 0, "y": 0.5, "z": 0}}
)

# 原地旋转
await pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Move"],
     "parameter": {"x": 0, "y": 0, "z": 1.0}}
)
```

## 二、MCF 模式指令 (SPORT_CMD_MCF)

## 三、运动模式切换

话题：`rt/api/motion_switcher/request`

| api_id | 参数 | 说明 |
|--------|------|------|
| 1001 | 无 | 查询当前运动模式 |
| 1002 | `{"name": "normal"}` | 切换到 Normal 模式 |
| 1002 | `{"name": "ai"}` | 切换到 AI 模式 |
| 1002 | `{"name": "mcf"}` | 切换到 MCF 模式 |

三种模式说明：

| 模式 | 说明 |
|------|------|
| `normal` | 标准四足运动模式 |
| `ai` | AI 智能模式（支持倒立等高级动作） |
| `mcf` | 多控制框架（最多动作，api_id 空间不同） |

```python
# 1. 查询当前模式
resp = await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["MOTION_SWITCHER"],
    {"api_id": 1001}
)
data = json.loads(resp['data']['data'])
current_mode = data['name']

# 2. 切换到目标模式
await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["MOTION_SWITCHER"],
    {"api_id": 1002, "parameter": {"name": "mcf"}}
)
await asyncio.sleep(5)  # 等待机器人完成切换
```

## 四、避障控制

话题：`rt/api/obstacles_avoid/request`

| API | api_id | 参数 | 说明 |
|-----|--------|------|------|
| `SWITCH_SET` | 1001 | `{"enable": true/false}` | 开启/关闭避障 |
| `SWITCH_GET` | 1002 | 无 | 查询避障状态 |
| `MOVE` | 1003 | `{"x":, "y":, "yaw":, "mode": 0}` | 避障模式下移动 |
| `USE_REMOTE_COMMAND_FROM_API` | 1004 | `{"is_remote_commands_from_api": bool}` | 启用 API 远程指令 |

```python
# 开启避障
await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["OBSTACLES_AVOID"],
    {"api_id": 1001, "parameter": {"enable": True}}
)
```

## 五、发送指令的标准格式

### 5.1 通用 RPC 格式

```python
from unitree_webrtc_connect import (
    UnitreeWebRTCConnection, WebRTCConnectionMethod,
    RTC_TOPIC, SPORT_CMD, SPORT_CMD_MCF,
)

# 无参数动作
await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Dance1"]}
)

# 有参数动作
await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["Move"],
     "parameter": {"x": 0.5, "y": 0, "z": 0}}
)

# 布尔参数
await conn.datachannel.pub_sub.publish_request_new(
    RTC_TOPIC["SPORT_MOD"],
    {"api_id": SPORT_CMD["BackFlip"],
     "parameter": {"data": True}}
)
```

### 5.2 完整连接示例

```python
import asyncio
from unitree_webrtc_connect import UnitreeWebRTCConnection, WebRTCConnectionMethod
from unitree_webrtc_connect.constants import RTC_TOPIC, SPORT_CMD

async def main():
    conn = UnitreeWebRTCConnection(
        WebRTCConnectionMethod.LocalSTA,
        ip="192.168.12.1",
        aes_128_key="7b7ad05fae7b79f3c0135f7417f895d0"
    )
    await conn.connect()

    # 向前走
    await conn.datachannel.pub_sub.publish_request_new(
        RTC_TOPIC["SPORT_MOD"],
        {"api_id": SPORT_CMD["Move"],
         "parameter": {"x": 0.3, "y": 0, "z": 0}}
    )
    await asyncio.sleep(2)

    # 打招呼
    await conn.datachannel.pub_sub.publish_request_new(
        RTC_TOPIC["SPORT_MOD"],
        {"api_id": SPORT_CMD["Hello"]}
    )

    await asyncio.sleep(3600)

asyncio.run(main())
```
